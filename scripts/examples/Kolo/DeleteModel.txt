8. DeleteModel
Purpose:
This command permanently deletes the modelâ€™s training output and associated files from the container. It is a more destructive action than uninstalling.

Key Details:

Overview:
When you are sure you no longer need the training output or fine-tuned model files, use this command to clear up storage space and remove the model completely.

Command:
./delete_model.ps1 "GodOutput" -Tool "unsloth|torchtune"
What Happens:
The script removes the directory (GodOutput) and all related files, ensuring that no trace of the model remains on the container.

Usage Note:
Use this command with caution as it permanently deletes data.
