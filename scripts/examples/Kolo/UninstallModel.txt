7. UninstallModel
Purpose:
This command removes an installed model from the active list in the Kolo environment without deleting its underlying files.

Key Details:

Overview:
Use this command when you want to temporarily remove a model from service, perhaps to free up resources or to avoid cluttering the active model list.

Command:
./uninstall_model.ps1 "God"
What Happens:
The specified model (God in this example) is unregistered from the system, meaning it will no longer appear as available on the web interface. However, the modelâ€™s data remains on the system unless explicitly deleted.
